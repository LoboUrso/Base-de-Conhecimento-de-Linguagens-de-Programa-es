# 🔍 Base de Conhecimento Tech

Uma aplicação web para pesquisar e explorar linguagens de programação, frameworks, ferramentas e tecnologias do universo dev.

---

## 💡 Sobre o Projeto

Digite o nome de uma tecnologia ou palavra-chave no campo de busca e veja instantaneamente informações como ano de criação, descrição e link para a documentação oficial. O projeto conta com mais de 70 tecnologias cadastradas.

---

## ✨ Funcionalidades

- **Busca em tempo real** por nome ou descrição da tecnologia
- **Cards informativos** com nome, ano de criação, descrição e link para a documentação oficial
- **Interface responsiva** adaptada para desktop e mobile
- **Design escuro** inspirado nas ferramentas de busca modernas

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** — estrutura da página
- **CSS3** — estilização com variáveis CSS, Flexbox e responsividade
- **JavaScript (Vanilla)** — lógica de busca e renderização dinâmica dos cards
- **JSON** — base de dados local com as tecnologias cadastradas
- **Google Fonts** — tipografia com a fonte Quicksand

---

## 📁 Estrutura do Projeto

```
├── index.html       # Estrutura principal da página
├── style.css        # Estilos e responsividade
├── script.js        # Lógica de busca e renderização
└── data.json        # Base de dados com as tecnologias
```

---

## 🚀 Como Rodar Localmente

Por utilizar `fetch()` para carregar o `data.json`, o projeto precisa ser servido por um servidor local. A forma mais simples é utilizando a extensão **Live Server** no VS Code:

1. Clone o repositório
2. Abra a pasta no VS Code
3. Clique em **"Go Live"** no canto inferior direito
4. O projeto abrirá automaticamente no navegador
